ROI & Incremental CLV calculation

- Incremental CLV (per user) = CLV_treated - CLV_control
- Total incremental value = sum(Incremental CLV for targeted users)
- ROI = (Total incremental value - Campaign cost) / Campaign cost

Notes:
- Use ATT when measuring effect on treated; use ATE for population-level policy planning.
- Report both relative % lift and absolute $ impact for leadership.
